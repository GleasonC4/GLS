# This Keyword

Keyword for an object to refer to itself.

## Commands

### `this`

`this`

Outputs the given languages equivilent to this.

## Usage

```
this
```

### CSharp

```csharp
this
```

### Java

```java
this
```

### Python

```python
self
```

### Ruby

```ruby
self
```

### TypeScript

```typescript
this
```
